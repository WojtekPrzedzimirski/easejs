GNU ease.js can be downloaded in a variety of formats, depending on the environment in which it will be used.

  • Production - Minified, 7.1 kB gzipped (25 kB uncompressed)
  • Development - Unminified, 30 kB gzipped (137 kB uncompressed)

npm

ease.js is available via npm for use with Node.js. Alternatively, one can simply clone the repository and use that directly.
$ npm install easejs
